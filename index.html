Hello, World! (From Node.js)

This is a simple Node.js server demonstrating basic routing.

We're serving different HTML pages based on the URL:

  • / for this homepage (index.html)
  • /about for the about page (about.html)
  • /contact-me for the contact page (contact-me.html)
  • Anything else leads to a friendly 404 (404.html)

It's a basic setup, but it's a start!
